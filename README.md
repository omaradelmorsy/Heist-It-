# Heist-It-
Our proposed project is Heist-It!, a game that tests your robbery skills under high pressure. We will have a locked safe that the thief (user) is trying to break in to steal the diamond. There will be a series of commands: Dial-It, Fire-It, and Jiggle-It

Hardware Files
The hardware files are in 2 sections. One has the Altium project that was used to create the project. The other is a zip of the Gerber files used for production

Software Files
The software file is the code used in the programming of the microcontroller in the system

Enclosure Files
The enclosure files are files that can be uploaded to Bambu studios and ready to be printed
