# Heist-It-
Heist-It! is a fast-paced game inspired by the Bop-It! toy, reimagined as a safe cracking heist experience. The player takes on the role of a thief attempting to break into a safe to steal a diamond. The game presents three randomized commands: Dial It!, Fire It!, and Jiggle It! and the player must perform the correct physical action before the timer expires. As the player scores points, the time allowed per command decreases, increasing the difficulty. The game ends either when the player reaches a score of 99 (triggering the safe to unlock and the diamond to be revealed) or when the player runs out of time or performs the wrong action (triggering a game over sequence). The system is built around an ATmega328P microcontroller and integrates multiple sensors, actuators, audio output, and a display, all housed in a custom-built enclosure designed to look like a miniature bank safe.
